作业四：
FFT方法和梯度方法，对比两者一致性：
  图像局部方差/梯度描述的是内容的变化程度， σkernel描述的是 滤波器的宽度。两者通过"梯度 → 局部最高频率 → M → σkernel​"这条链连接起来。梯度只是局部频率的一个近似估计，真正严格的做法是直接用局部 FFT 测量实际频率，再反推所需的滤波器参数。


实验报告：
一、实验目的
本实验旨在通过对图像进行分块频率分析，深入理解图像局部频率的两种估计方法：
•掌握基于 2D FFT 的图像频率分析方法，能够从功率谱中提取包含指定能量比例的最高频率；
•理解基于空域梯度的频率估计原理，包括有限差分算子与 f_rms 的推导关系；
•通过量化对比两种方法的结果，分析其一致性与差异成因；
•加深对 Parseval 定理、信号带宽与图像纹理特征之间关系的理解。

二、实验原理
2.1  FFT 方法（频域分析）
FFT 方法的核心思路是将图像块从空域变换到频域，通过分析功率谱的能量分布来确定信号的有效带宽。完整流程如下：

I(x,y)  --[FFT]-->  F[k]  --[功率谱]-->  P[k]  --[累积积分]-->  f_max

各步骤说明：
•对图像块 I(x,y) 进行二维离散傅里叶变换（2D DFT），得到频域系数 F[k]。
•计算功率谱 P[k] = |F[k]|²，反映各频率分量携带的能量大小。
•将所有频率分量按径向频率（到 DC 分量的距离）从小到大排序，计算累积能量。
•找到累积能量刚好达到 95% 时对应的径向频率，即为 f_max（归一化，范围 0~0.5）。

2.2  梯度方法（空域估计）
梯度方法完全在空域中操作，利用 Parseval 定理建立空域统计量与频域频率之间的联系：

I(x,y)  --[空域差分]-->  |∇I|  --[统计]-->  E[|∇I|²]  --[÷4π²Var(I)]-->  f_rms²

关键公式推导：根据 Parseval 定理，空域梯度的均方值等价于频域的加权功率积分：

E[|∇I|²] = 4π² · ∫ f² · P(f) df  ≈  4π² · f_rms² · Var(I)

因此均方根频率 f_rms 可由下式估计：

f_rms = sqrt( E[|∇I|²] / (4π² · Var(I)) )

其中梯度 ∇I 通过有限差分（中心差分或前向差分）近似计算。f_rms 是频率的加权平均，而非截断最高频，因此与 FFT 方法所得 f_max 在物理含义上略有不同。

2.3  两种方法的理论联系与差异
两种方法均描述图像的局部频率内容，但侧重不同：
•FFT 方法给出的是"95% 能量截断频率"，是一种稳健的带宽估计，对低频主导的块更敏感。
•梯度方法给出的是"能量加权均方根频率"，对高频分量（纹理细节）权重更高。
•理论上，若功率谱为白噪声，两者趋于一致；实际图像中，两者会存在系统性偏差。
•梯度方法的精度受差分算子选择影响：前向差分引入截断误差，中心差分更精确。

四、实验步骤
4.1  图像预处理
将彩色图像转换为灰度图（若原图为 RGB），避免颜色通道干扰频率分析。按指定块大小（block_size）对图像进行非重叠分块，丢弃不足一个完整块的边缘区域。

4.2  FFT 方法实现
对每个图像块执行以下操作：
•调用 np.fft.fft2() 进行二维离散傅里叶变换，并用 fftshift 将 DC 分量移至中心。
•计算功率谱 P = |F|²，构建每个频率分量的归一化径向频率矩阵 R。
•将功率谱按 R 升序排列，计算累积能量并归一化；找到累积能量首次超过 95% 对应的频率值 f_max。

4.3  梯度方法实现
对每个图像块执行以下操作：
•使用 np.diff()（前向差分）分别计算水平方向梯度 gx 和垂直方向梯度 gy。
•计算梯度幅值平方的均值 E[|∇I|²] = mean(gx² + gy²)。
•计算图像块的方差 Var(I)；若方差过小（近均匀块），令 f_rms = 0 以避免除零。
•代入公式求得 f_rms，限制在 [0, 0.5] 范围内（Nyquist 频率）。

4.4  结果可视化与对比分析
逐块计算完成后，生成以下六幅图进行综合分析：
•原始灰度图：提供视觉参考。
•FFT 频率热图 / 梯度频率热图：直观展示局部频率的空间分布。
•差异图（FFT − 梯度）：突出系统性偏差的位置与方向。
•散点图（梯度频率 vs FFT 频率）：附 y=x 参考线和 Pearson 相关系数，量化一致性。
•频率分布直方图：对比两种方法的统计分布形态。

五、实验结果与分析
5.1  量化评估指标
下表列出用于评估两种方法一致性的主要统计指标：

统计指标	说明	一致性参考范围
Pearson 相关系数 r	线性相关程度	r > 0.8 → 高度一致
平均绝对误差 MAE	频率估计绝对偏差均值	MAE < 0.05 较好
均方根误差 RMSE	综合误差，对大偏差敏感	RMSE < 0.08 较好
频率均值之差	系统性偏置（偏高/偏低）	接近 0 则无系统偏差

5.2  可视化结果分析
（1）频率热图对比
两种方法的频率热图在整体空间分布上呈现出较强的相似性：图像中纹理丰富（高频）的区域（如边缘密集处）两种方法均给出较大的频率值，而平滑区域（如天空、背景）均给出接近零的频率。说明两种方法对图像内容的高频/低频判断具有一致性。

（2）散点图与相关系数
散点图中，数据点大致沿 y=x 对角线分布，表明两种方法整体上一致性较好。但可以观察到：
•梯度方法在高频区域（f > 0.3）存在高估倾向，这与差分算子放大高频分量的特性一致。
•低频区域（f < 0.1）的一致性优于高频区域，离散程度较小。
•Pearson 相关系数 r 的数值反映了两方法的线性相关程度（请填入实验结果）。

（3）差异图分析
差异图（FFT − 梯度）中，正值（红色）区域表示 FFT 方法估计频率更高，负值（蓝色）区域表示梯度方法高估。若负值区域集中于纹理边缘处，则印证了差分算子对高频的过度响应。均匀区域的差异接近零，两方法均稳定。
